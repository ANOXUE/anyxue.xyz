Python简介

先简单说一下python，他由荷兰人Guido van Rossum于1989年发明，第一个公开发行版发行于1991年。是一门面向对象的解释型程序设计语言，具有开发效率高，跨平台，简单易学等优点。从设计之初就致力提升代码的可阅读性，崇尚优美、简单和清晰。强制缩进是python的一大特色，这点在3.x源码中有明显的体现。而且由于python易于嵌入和移植的特性，使他非常适合作为一种脚本语言，除了应用于开发领域，还能很好地契合运维和网络安全等领域的相关需求。

由于身为解释型语言，python更适合开发应用程序。应用python的知名企业：Youtube、Dropbox、BT、Quora（中国知乎）、豆瓣、知乎、Google、Yahoo!、Facebook、NASA、百度、腾讯、汽车之家、美团等。

python目前主要有四种解释器：

  • Cpython：官方推荐解释器。可以转化成C语言能识别的字节码
  • Jython: 可以转化成Java语言能识别的字节码
  • IronPython：可以转化成.net语言能识别的字节码
  • PyPy: 动态编译，可以显著提高Python代码的执行速度

基础知识

基于python3

变量命名规则
  • 全部书由数字、字母和下划线组成
  • 不能以数字开头
  • 不能是python内置语法函数
  • 不能使用中文（但是有拼音啊~ 有兴趣的童鞋还可以了解一下”草莽“）
  • 不能过长
  • 要具有描述性
常量

在python中，用名称大写的变量表示常量

注释
  • 单行注释符：#
  • 多行注释符：’’’neirong’’’ 或者 “””内容”””
数据类型
  • int（整数型）：值为整数
  • float（浮点型）：值含小数点
  • str（字符型）：值为任意字符串，且需加引号
  • bool（布尔型）：值为True 或 False
